## 🔴 发现的问题

### 1. 语言检测不准确

| 问题 | 详情 |
|------|------|
| 简体中文误检测为粤语 | zh_en_1, zh_en_2, zh_en_3 的 detected_source 都是 "yue" |
| 日语误检测为粤语 | ja_en_1, ja_en_2, ja_en_3 的 detected_source 都是 "yue" |
| 韩语误检测为粤语 | ko_en_1, ko_en_2, ko_en_3 的 detected_source 都是 "yue" |

**影响**: 虽然语言检测不准确，但大部分翻译结果仍然正确，说明模型的翻译能力强于语言检测能力。

### 2. 粤语输出问题

英语翻译到粤语时，输出的是书面中文而非粤语口语：
- en_yue_1: 输出「我快要遲到了」而非「我就嚟遲到」
- en_yue_2: 输出「請您能說慢一點嗎」而非「可唔可以講慢啲」

### 3. 翻译失败案例

- **ko_en_1**: 模型输出了解释而非翻译，这是严重的功能性错误

### 4. 输出异常

- **fr_en_3**: 输出包含大量异常的点号字符

### 5. 翻译不完整

- **ja_en_3**: 丢失了道歉部分「申し訳ございませんが」
